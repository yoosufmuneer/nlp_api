# nlp_api

###### 1. install python3 & django2

###### 2. Install NLTK

`pip install -U nltk`

###### 3. Install TextBlob

`pip install -U textblob`

###### 4. Go To The Directory

`cd nlp_api`

###### 5. Run the API

`python manage.py runserver`
